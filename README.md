

# My Portfolio Resume

## Description
**My Portfolio Resume** is a single-page resume website built using pure semantic HTML.
The project focuses on clean structure, accessibility, and readability without relying on CSS or JavaScript.
It is designed to be screen-reader friendly and easy to evaluate.
